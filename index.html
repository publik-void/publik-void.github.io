PhD Objective

  • Robust assessment of motor performance in genetic ataxias
  • Based on body-worn sensor recordings in real-life settings
  • Sensitive to moderate-term change in a single patient

Spinocerebellar Ataxia (SCA)

  • Inherited neurodegenerative disease
  • Affects the cerebellum and spinal cord
  • Rare disease: ≤5 people per 100,000
  • Increased gait variability
  • Widened base of support
  • Impaired foot placement
  • Loss of trunk control

Gait features

  • Currently ~50 features (+ ~2000 transformations)
  • High redundancy

Available data

  • Main dataset: gait recordings from Tübingen
    • 27 spinocerebellar (pre-)ataxia patients
    • 36 healthy controls
    • Lab-based & real-life
    • Yearly follow-ups over several years
  • Existing lab-based movement recordings (PROSPAX)
    • ≥25 spastic ataxia patients
    • ≥50 healthy controls
  • Planned prospective study
    • ≥10 ataxia (SCA or AT) patients

Discussion

Composite features

  • Plan: Find the optimal combination of features
  • That fits the expected longitudinal delta the closest
    • (controls: 0/year, patients: 1/year)
  1. Reduce feature space (e.g. factor analysis)
  2. Least-squares / SVM / k-NN / …

Latent variable models

  • Plan: Model the disease progression as a latent variable
  • Model predictions can serve as a test for treatment effect
Candidate methods:
  • Longitudinal Item Response Theory
  • Hierarchical Bayesian Time-Series Models

Context sensitivity

Plan: Compute context proxies for every walking bout:
  • Macroscopic characteristics of the walking bout
    • Gait speed, frequency of turns
  • Time of day
  • Weather
  • …
And factor them in:
  • Either: Compute features over strata of context variables
  • Or: Model context variables as fixed effects

Further gait features

Plan: Find further gait features sensitive to ataxia
  • Polynomial transforms for overall foot trajectory
  • Jerk score
  • …?
